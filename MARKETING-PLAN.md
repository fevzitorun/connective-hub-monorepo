## 7. Referral & Viral Mekanikler

### NPS Döngüsü

```
Kullanıcı → İlan paylaş (WhatsApp / Instagram)
         → Paylaşıma tıklayan → 7fil'e gelir
         → "Bu ilanı 7fil'de gördüm" tracking
         → İlan sahibine bildirim: "İlanınız 47 kez paylaşıldı"
```

### Sosyal Kanıt Motoru

```
Her ilana otomatik eklenen:
  • "Bu ilan hukuki doğrulamadan geçti" badge
  • "Atlas AI değerleme: Piyasanın %8 altında" badge
  • Görüntülenme sayacı: "Son 24 saatte 128 kez görüntülendi"
  → FOMO yaratır → daha fazla sorgu
```

### Viral Hook'lar

```
ATLAMA HOOK'U:
  İlan paylaş → paylaşılan link özel tracking URL
  → Arkadaş kayıt olursa → ilan sahibi "Super Görünürlük" kazanır (7 gün ücretsiz)

DEĞERLEME HOOK'U:
  "Evinizin değerini öğrenin — 30 saniye, ücretsiz"
  → E-posta al → lead magnet
  → FILTERRA değerleme sonucu e-postayla gönderilir

SERTIFIKA HOOK'U:
  Satıcı sertifika alıyor → sosyal medyada paylaşıyor
  → "7fil Onaylı Mülk" organic reach
```
